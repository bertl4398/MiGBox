Module sync

source code

Synchronization methods and classes to synchronize between file system abstractions see MiGBox.FileSystem.

Classes [hide private]
  EventQueue
This class is used to keep track of the file system events and is as for now the default python queue.
  EventHandler
This class handles all events observed from the watchdog observer by putting them into an event queue, that will be processed by the synchronization thread.
