Function Details [hide private]

run(host, port, hostkey, userkey, rootpath, backlog=0, logfile=None, loglevel=None)

source code 

Main entry point to run the sftp server.

Parameters:
  • host (str) - server's name or ip address.
  • port (int) - server's port number for listening.
  • hostkey (str) - path to the server's private rsa key.
  • userkey (str) - path to the user's public rsa key.
  • rootpath (str) - path to be served via SFTP.
  • backlog (int) - maximum number of queued connections.
  • logfile (str) - path to the server's log file.
  • loglevel (str) - log level, usually 'INFO' or 'DEBUG'
