write_config(configfile, values, server=False)

source code 

Write the given values to the given configfile.

Values are expected to be in the form of

{'section': {'option': value}}

Writes always the default structure.

Parameters:
  • configfile (str) - path to the config file.
  • values (dict) - configuration values.

read_config(configfile, server=False)

source code 

Read the given configfile and return a dictionary with all sections, options and values in the form of

{'section': {'option': value}}

A default configuration file is read first to set the right structure.

Parameters:
  • configfile (str) - path to the config file.
Returns: dict
dictionary of all values.
