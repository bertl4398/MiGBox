sync_file(src, src_path, dst, dst_path)

source code 

Synchronize a file from src file system abstraction to dst file system abstraction from src_path to dst_path.

If the files given by src_path and dst_path are equal, nothing is done.

Parameters:
  • src (MiGBox.FileSystem) - source file system abstraction.
  • src_path (str) - source path.
  • dst (MiGBox.FileSystem) - destination file system abstraction.
  • dst_path (str) - destination path.
